Exemplo de Formulário no Google Apps Script

Este HTML pode ser usado para criar formulários dentro do Google Apps Script, como por exemplo, em uma planilha do Google Sheets.

Para abrir o formulário dentro da interface do Google Sheets, utilize o código abaixo: 

      function openForm() {
        var html = HtmlService.createHtmlOutputFromFile('form/index')
        .setWidth(400)
        .setHeight(300);
      SpreadsheetApp.getUi().showModalDialog(html, 'Formulário');
  }

Esse código abrirá um modal exibindo o formulário.
